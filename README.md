# RPS
Description

This project is an interactive AI-based game that uses computer vision and deep learning to recognize hand gestures in real time. A Convolutional Neural Network (CNN) classifies gestures captured via webcam and plays Rock–Paper–Scissors against the user.

 Features

Real-time hand gesture detection using webcam

Image classification using CNN

Supports Rock, Paper, Scissors prediction

Live game interaction with computer opponent

Displays prediction results and game outcome

Lightweight model for smooth performance

 Tech Stack

Language: Python

Libraries: TensorFlow / Keras, OpenCV

Concepts: Deep Learning, Computer Vision

⚙️ Installation & Setup

Clone the repository

git clone <your-repo-link>

Navigate to the project folder

cd rock-paper-scissors-cnn

Install dependencies

pip install -r requirements.txt

Run the project

python main.py
 Project Structure

/dataset → Training images

/model → Trained CNN model

/src → Main application code

/utils → Helper functions
